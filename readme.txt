== Description ==

[Demo](https://tishonator.com/plugins/slipry-slider)

Configure slideshows (titles, texts, and images) in minutes and display them as a Responsive Slider in your website by inserting a shortcode.


= Shortcodes: =

Insert `[slipry-slider]` on a page to display a slider with data configured in Admin Panel -> Left Menu -> SliprySlider Page. 

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `slipry-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Slipry Slider' -> fill Slider fields
4. Insert the shortcode [slipry-slider] to display Slider to any page or post
5. Enjoy!
